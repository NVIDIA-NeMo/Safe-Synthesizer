# Citation Guide

## To Cite the Project

If you use the Project in a publication, please use citations in the following format (BibTeX entry for LaTeX):

```tex
@Manual{,
  safe-synthesizer,
  title = {Safe Synthesizer: A package to create synthetic data from existing data with privacy guarantees)
  author = {Aaron Gonzales, Kendrick Boyd, Sean Yang, Nina Xu, Marjan Emadi},
  year = {2026},
  url = {https://github.com/nvidia-nemo/safe-synthesizer},
}
```
